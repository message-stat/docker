# Дистрибутив VKStat

Найти можно тут: [vkstat.soprachev.com](https://vkstat.soprachev.com/analytics)


## Запуск

### Скачать

```sh
git clone https://github.com/message-stat/docker.git
```

### Запустить

```sh
docker compose up
```

Открыть в браузере `http://localhost:5005` (рекомендуется Google Chrome)

### Остановить

```
ctrl + c
```
